### API

#### Getting Started

Basic link `api.trustybot.xyz`

As bot is not available api is not available too!


#### What can I fetch with API

You can fetch:
- Server Statistics 
```javascript
GET api.trustybot.xyz/servers/server_id/all
```
- Bot Statistics 
```javascript
GET api.trustybot.xyz/bot/all
```
- Level Statistics 
```javascript
GET api.trustybot.xyz/servers/server_id/level
```
- Bans
```javascript
GET api.trustybot.xyz/servers/server_id/bans
```
